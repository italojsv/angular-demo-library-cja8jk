Client-Side AuthZ Bypass
Admin

This demo contains a link to the administrative interface; however, this "Admin" page is restricted to administrative users. We can bypass the client-side access controls implemented by Angular, allowing us to access the "Admin" page.

Tip

We can use the FireFox Web Developer Debugger to view the Angular source files. For this demo, we would use the Debugger to navigate to the app.admin.html and app.admin.ts files where we could view the contents of these files and obtain sensitive data.
As shown below, the "Admin" page contains an AJAX request to the "/admin/addUser" endpoint with an embedded "Authorization" header. An unauthorized user can extract the endpoint and authorization token from the page source and submit requests to the endpoint directly, bypassing client-side controls.
